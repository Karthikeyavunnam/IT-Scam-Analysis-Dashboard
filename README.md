I recently worked on an **IT Scam Analysis Dashboard** using MS Excel and Power BI, where I explored real-world scam data to understand patterns, trends, and key risk areas.

The main goal of this project was to turn raw data into meaningful insights. I started with data cleaning and preprocessing in Excel, then built an interactive dashboard in Power BI. Using basic DAX measures, I calculated important KPIs such as total cases, total financial loss, and category-wise breakdowns.

Through this project, I learned how powerful data visualization can be in identifying trends like region-wise scam distribution, year-wise growth, and the most common scam types. It also helped me improve my understanding of data modeling and dashboard design.

This experience strengthened my interest in data analytics and showed me how data-driven insights can help in decision-making.
